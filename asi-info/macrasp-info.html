Resumen

En esta práctica voy a detallar cómo fue la configuración inicial de los siguientes dos equipos: una Raspberry Pi y un Mac Mini. Describiré, paso a paso, todo lo que hicimos para dejar ambos sistemas operativos listos. Esto incluye desde la selección del idioma y la conexión a la red, hasta la creación de usuarios y la realización de las actualizaciones necesarias.

Portada

Configuración Raspberry Pi

1. Inicio y Región

Lo primero fue conectar la Raspberry y encenderla. Al encenderla por primera vez, aparece el asistente de inicio que nos ayudará a configurar el sistema. Al darle a siguiente, nos pedirá que seleccionemos nuestro país, idioma y zona horaria.

Set Country

2. Creación de Usuario

Después de darle a continuar, deberemos crear nuestro usuario con su contraseña, siguiendo las instrucciones del profesor, le asignamos "aulaateca26" tanto de usuario como de contraseña.

Create User

3. Conexión WiFi

Al crear el usuario, lo siguiente que nos aparece es para conectarnos a la red WiFi.

Select WiFi

4. Configuración del Navegador

Una vez nos hemos conectado a la red WiFi nos pregunta qué navegador queremos usar por defecto junto con una opción para desinstalar el que no usemos. Seleccionamos Chromium y NO marcamos la opción para desinstalar Firefox.

Choose Browser

5. Actualización del Sistema

A continuación, nos pide actualizar, marcamos que si queremos actualizar ahora y esperamos.

Update Software

6. Depósito de Claves

Una vez se han completado las actualizaciones, reiniciamos el equipo e introducir una contraseña para el depósito de claves. Siguiendo las instrucciones del profesor, esta contraseña también será "aulaateca26".

Depósito de claves

Configuración Mac Mini

1. Idioma y Región

Lo primero que debemos hacer es conectarlo y encenderlo. Una vez encendido, aparecerá el asistente que nos guiará en la configuración inicial. La primera opción que nos aparece es para seleccionar el idioma, por lo que seleccionamos español. Después nos consulta por nuestra región, donde seleccionamos España.

Idioma

2. Tratamiento Personal

A continuación, nos pide que seleccionemos como quiere que se nos trate, si como masculino, femenino o neutro. Esto no tiene mucha importancia, pero seleccionamos neutro y le dimos a siguiente.

Tratamiento

3. Accesibilidad

Después nos aparecen ajustes de accesibilidad, investigamos sobre que iba cada uno, pero finalmente no seleccionamos nada y le dimos a continuar.

Accesibilidad

4. Conexión WiFi

Una vez omitido el paso de accesibilidad, nos pedía conectarnos a una red WiFi, por lo que nos conectamos a la misma red que con la Raspberry Pi.

Red WiFi

5. Asistente de Migración

Cuando nos conectamos, nos apareció un asistente de migración, como no queríamos migrar ningún tipo de dato pulsamos en "Ahora no".

Migración

6. Cuenta de Apple

Posteriormente nos pedía que iniciaramos sesión con nuestra cuenta de Apple, seleccionamos la opción "Configurar más tarde".

Apple ID

7. Términos y Condiciones

Después nos pidió que aceptáramos los términos y condiciones, los cuales aceptamos para poder utilizar el equipo.

Términos

8. Creación de Cuenta

Luego tuvimos que crear nuestro usuario, por lo que, siguiendo las instrucciones del profesor, volvió a ser "aulaateca26" tanto para el usuario como la contraseña.

Crear Usuario

9. Configuración de Servicios y Cierre

Después nos pidió activar varias opciones, como la localización, zona horaria, enviar datos de análisis, y una opción para confirmar si queremos recibir informes sobre el tiempo de uso. Esta última opción marcamos en "Configurar más tarde" aunque el cursor aparezca en continuar. Una vez marcadas las opciones de los pasos anteriores, reiniciamos el equipo y ya estaba listo para usar.

Localización

Análisis

Tiempo de uso
