Configuración inicial Jetson Nano y LM Studio

Guía paso a paso sobre cómo preparar una NVIDIA Jetson Nano. Aquí explico desde cómo conectar los cables y configurar Ubuntu, hasta cómo montar un disco SSD y dejar listo LM Studio para usar Inteligencia Artificial.

Resumen

En esta práctica lo primero que hacemos es preparar todo el material de la Jetson Nano y asegurarnos de que enciende correctamente. Después pasamos a configurar el sistema operativo paso a paso, eligiendo el idioma, conectándonos a la red del aula y creando nuestro usuario.

Una parte importante es cuando trabajamos con la terminal para montar un disco SSD externo, usando varios comandos para dejarlo listo. Por último, instalamos las librerías necesarias y descargamos LM Studio para probar un modelo de Inteligencia Artificial de 4 billones de parámetros.

1. Preparación de Hardware y Primer Arranque

Verificación Física y Encendido

Lo primero que debemos hacer es preparar el material. Nos aseguramos de tener la Jetson Nano y sus cables a mano. A continuación, conectamos la Jetson a la corriente y al monitor para comprobar que funciona. Una vez encendida, nos pide aceptar las licencias y después seleccionamos el idioma.

5. Preparación del Entorno y Montaje SSD

Configuración de Almacenamiento Externo

Para instalar LM Studio, seguimos la guía de NVIDIA. Abrimos la terminal y usamos `lspci` y `lsblk` para revisar si tenemos montado el disco. Luego usamos comandos para formatearlo (`sudo mkfs.ext4`) y montarlo en la carpeta `/ssd`. También editamos el fichero `/etc/fstab` para que se quede guardado. Mientras hacemos esto, dejamos descargando LM Studio de la página oficial.

6. Instalación y Prueba de LM Studio

Despliegue de IA Generativa

Instalamos libfuse2 y damos permisos de ejecución a la AppImage de LM Studio para poder instalarlo. Una vez instalado, lo abrimos y vamos a la lupa para buscar modelos de IA. Instalamos el modelo "qwen2-vl-4b", donde "4b" significa 4 mil millones de parámetros. Para terminar, creamos un chat nuevo, escribimos "hola" y nos fijamos en los tokens por segundo para ver qué tal funciona.
