Configuración inicial Jetson Nano y LM Studio

Documentación técnica sobre el despliegue inicial de una NVIDIA Jetson Nano. El proyecto abarca desde la conexión de hardware y configuración del sistema operativo (Ubuntu) hasta la implementación de almacenamiento externo y la instalación de software de Inteligencia Artificial Generativa (LM Studio).

Resumen

El objetivo de esta práctica fue poner en marcha un dispositivo de computación en el borde (Edge Computing) para tareas de IA. [cite_start]El proceso comenzó con la preparación del hardware Jetson Nano y su cableado[cite: 119, 120]. [cite_start]Posteriormente, se realizó la configuración "out-of-the-box" del sistema operativo, incluyendo localización, red y creación de usuarios administrativos[cite: 121, 126].

[cite_start]Una fase crítica consistió en el montaje técnico de una unidad SSD externa mediante comandos de terminal para expandir el almacenamiento[cite: 146, 154]. [cite_start]Finalmente, se procedió a la instalación de las librerías necesarias (libfuse) para ejecutar LM Studio y descargar un modelo de lenguaje (LLM) de 4 billones de parámetros para realizar pruebas de inferencia[cite: 160, 163, 166].
