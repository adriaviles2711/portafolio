Resumen

Plataforma de gestión de guías de trofeos diseñada para ayudar a los jugadores a organizar su camino hacia el platino. La aplicación permite crear, editar y visualizar estrategias detalladas, calculando automáticamente estadísticas de dificultad y alertando sobre trofeos perdibles en una interfaz accesible y adaptada a móviles.

Fue desarrollada íntegramente con HTML, CSS y JavaScript puro (vanilla), funcionando como una Single Page Application (SPA). Destaca por su implementación de IndexedDB para el almacenamiento local persistente de datos e imágenes, eliminando la necesidad de bases de datos externas o conexión a internet.

Características principales

  • Persistencia Local con IndexedDB: Implementa un sistema de almacenamiento robusto que guarda guías completas e imágenes en Base64 directamente en el navegador, permitiendo un uso totalmente offline y privado.
  • Gestor de Contenido Dinámico: Ofrece un editor completo para crear fichas de juegos, calcular estadísticas (horas, dificultad) y gestionar listas de trofeos con metadatos específicos (tipo de medalla, si es perdible, etc.).
  • Lógica de Visualización Inteligente: La aplicación procesa los datos para generar automáticamente resúmenes de medallas y alertas de navegación rápida para los "trofeos perdibles", facilitando la estrategia del jugador.
  • Accesibilidad y Diseño Adaptable: Interfaz "Mobile First" con un módulo de ajustes dedicado que incluye modo oscuro, control de tamaño de fuente y reducción de movimiento para una experiencia inclusiva.

Este proyecto demuestra un dominio avanzado de las APIs del navegador (IndexedDB, FileReader), arquitectura SPA (Single Page Application) y manipulación eficiente del DOM sin dependencias externas.

Prompts Utilizados

Aquí se documentan los prompts enviados a la IA y los resultados que generaron, mostrando la evolución del proyecto desde la idea inicial hasta el resultado final.

Prompt 1: Versión Inicial y Estructura con IndexedDB

Actúa como un desarrollador con experiencia de varios años en el desarrollo de aplicaciones. Haz una app de guias de platinos de videojuegos en HTML, CSS y JavaScript puro (vanilla JavaScript) embebido, no uses frameworks para JavaScript ni para CSS. Las guías, junto con las imágenes subidas, se guardarán en local usando IndexedDB. La idea es que haya una sección donde apareceran todos las guias añadidas de los diferentes juegos, y al seleccionar uno, aparezca la guía. Quiero que al abrirlo aparezca la siguiente información del platino: Horas para conseguir el platino, dificultad (del 1 al 10), minimo de partidas para el platino, Nº trofeos online, Nº trofeos perdibles y que ponga si influye o no la dificultad en los trofeos. Está información la tendrá que poner el usuario a la hora de crear la guia. Luego aparecerá un miniresumen de cosas importantes, como por ejemplo, que dificultad debe jugar la persona para conseguir el platino, cuantos trofeos perdibles hay (y un hipervinculo a ellos, que estaran en la misma pagina, osea que solo será como un indice para que baje hasta el trofeo) y más información importante. Luego aparecerán los logros en el orden en el que el usuario los haya añadido, junto con una foto del logro, si es de bronce, plata u oro y s es perdible o no. Junto con la foto aparecerá el nombre y una descripción que explicará como conseguirlo, que también tendrá que poner el usuario (como todo). Para que el usuario añada cosas, añade un botón para crear una guía y otro dentro de las guías para editarlas. Para crearlas, será un formulario donde irá pidiendo foto, descripción, decir si es perdible o no, y poner si es bronce, plata, oro o platino. Hazlo como veas conveniente y si crees que me ha faltado algo, añadelo. Todo se guardará en un indexedDB en local, recuerda. Hazlo también compatible para móviles, y pon cosas de accesibilidad, como modo oscuro y más opciones de ajustes.

Prompt 2: Corrección de idioma y mejoras visuales

Una vez añadido un trofeo, aparece bronze en vez de bronce, lo mismo con platino, que aparece platinum. Ponlo en español. Además, añade al principio cuantos trofeos de bronce, plata, oro y platinos hay. También, haz los ajustes más bonitos y añade más funciones de accesibilidad. Es muy soso ahora mismo.
